# Eaz-Welfare  
NGO Community & Social Welfare Website

---

## Project Overview

Eaz-Welfare is a professionally developed NGO community website created as a **client project**. The purpose of the website is to provide a clear, accessible, and trustworthy digital presence for social welfare initiatives and community engagement.

This repository is publicly visible **only for demonstration and portfolio showcase purposes**.

Live Website:  
https://eaznexora.github.io/Eaz-Welfare/

---

## Project Objective

The primary objective of this project was to design and develop a responsive, clean, and informative website that effectively communicates an NGO’s mission, activities, and values.

Key objectives included:
- Establishing a professional digital identity  
- Ensuring ease of navigation and accessibility  
- Delivering a responsive experience across devices  
- Maintaining a lightweight and maintainable frontend codebase  

---

## Scope of Work

This repository includes:

- Frontend website design and development  
- Responsive layout implementation  
- Structured content presentation  
- Client-side interactivity  
- Static website deployment  

The following are **out of scope** for this repository:
- Backend systems or databases  
- Payment or donation processing  
- User authentication or admin panels

| Path / File        | Description |
